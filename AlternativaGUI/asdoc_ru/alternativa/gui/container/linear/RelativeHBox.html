Class RelativeHBoxProperties | Methods
 
Packagealternativa.gui.container.linear
Classpublic class RelativeHBox
InheritanceRelativeHBox Container GUIobject flash.display.Sprite

Горизонтальный контейнер с заданной шириной элементов. Если элементов больше, чем задано, оставшееся место делится между ними поровну. Если все размеры заданы в пикселях и осталось место — увеличится зазор между элементами.
