Packagealternativa.gui.container.list
Classpublic class List
InheritanceList GUIobject flash.display.Sprite
ImplementsIList
SubclassesDropDownItemContainer, Tree

Ленивый контейнер со списком элементов.

Отрисовывает только те элементы, которые видны в данный момент.

Позволяет выбрать только один элемент из списка.
