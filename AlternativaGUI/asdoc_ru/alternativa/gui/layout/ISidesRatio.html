Interfacepublic interface ISidesRatio

Интерфейс объекта имеющего предпочтительное соотношение сторон (высота/ширина).

Такой объект должен пересчитывать размеры по обеим сторонам при установке одного из них. Контейнер таких объектов должен пытаться сохранить это соотношение при задании размеров элемента.
