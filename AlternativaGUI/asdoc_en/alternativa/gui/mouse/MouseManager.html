Packagealternativa.gui.mouse
Classpublic class MouseManager
InheritanceMouseManager flash.display.Sprite

MouseManager class monitors mouse actions: mouseclick, mouseover, drag'n'drop, change the mouse cursor.

If GUIObject has listener of MouseEvent.CLICK event, then MouseManager will call it. If you need to use browse() method of FileReference class, then you must add event listener to the stage and monitor the mouse clicks. Or put Sprite into the object and add event listener to it.
