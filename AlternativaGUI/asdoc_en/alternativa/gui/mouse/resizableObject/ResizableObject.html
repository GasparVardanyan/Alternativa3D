Packagealternativa.gui.mouse.resizableObject
Classpublic class ResizableObject
InheritanceResizableObject GUIobject flash.display.Sprite
ImplementsIResizableObject

ResizableObject allows you to stretch or compress the object.

If _resizableContainer is set, then it will be rubber automatically.

For using of this class with your object, you must put an instance to it.
