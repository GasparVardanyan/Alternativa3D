Packagealternativa.gui.mouse
Classpublic class CursorManager

Cursor manager.

It registers cursors and controls display.

If the specified cursor was not found among the registered, it shows the default cursor: MouseCursor.AUTO.

To replace the existing cursors enough to call them the same.
