Classpublic class DataProvider
InheritanceDataProvider flash.events.EventDispatcher

At DataProvider class are some methods and properties, which are allow to ask and change the data at any list based component (List, DropDownList, DropDownMenu, Tree).

Data provider is linear collection of the elements, which are used for data source (eg, array). Each element in data provider is object, which contains one or more data fields. Elements contained in data provider, are able to call by specifying their index (DataProvider.getItemAt()).

sortOn()method 
public function sortOn(fieldName:String, order:String):*

Sorts items, which are contained in data provider, on specified field and send the DataChangeType.SORT event . Specified field can be a string or array of string value, which determine a few sortable fields in precedence order.

Parameters
fieldName:String — Field of the item that you want to sort. This value can be string or array of string values.
 
order:String — Options of the sorting.

Returns
* — Returned value depends on whether the method gets any arguments. For more information, see the description of the Array.sortOn() method. If sortOption property equal Array.UNIQUESORT, then this method returns 0.
