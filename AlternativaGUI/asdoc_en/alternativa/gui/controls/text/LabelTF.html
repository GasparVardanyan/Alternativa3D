Packagealternativa.gui.controls.text
Classpublic class LabelTF
InheritanceLabelTF GUIobject flash.display.Sprite
SubclassesText, TextInput

Text label with two modes, whick are switched by setDimensionByText flag. When setDimensionByText flag = true the size of label are depend only on the specified text. Otherwise, you can specify the width and alignment in a given area for the label.
