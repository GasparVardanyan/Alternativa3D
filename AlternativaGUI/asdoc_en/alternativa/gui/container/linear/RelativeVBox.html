Packagealternativa.gui.container.linear
Classpublic class RelativeVBox
InheritanceRelativeVBox Container GUIobject flash.display.Sprite

Vertical container with specified items width. If the number of items is greater than specified, the remaining space is divided equally between them. If all sizes specified in pixels, and there is a place - will increase space between the items.
