Packagealternativa.gui.container.tileList
Classpublic class TileList
InheritanceTileList Container GUIobject flash.display.Sprite

Tile list. You can specify cell optimal width. Width detection looks at the first item of container, if item width < optimal width, then it's given optimal, if item width > optimal, then optimal=item width. You can specify vertical align (columnAlign) and horizontal align (rowAlign) of items in cell.
