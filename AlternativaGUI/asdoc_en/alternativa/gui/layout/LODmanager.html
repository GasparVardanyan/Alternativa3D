Classpublic class LODmanager

Base class LODmanager.

It responsible for the detail of objects (LODs) in a scene. Manager send LOD index to all objects with ISimpleLODobject interface.

Numberof LOD (LODIndex) depends on height and width of the stage, which is passed from LayoutManager. Object can change the scins, data, remove / show different panels, etc., depending on the number of the LOD. After changing number LOD, you need to call draw method.

To change the boundaries of the index switching, you can use setters on limitsH and limitsV, which passed an array with the boundaries.

There is also a list LODlistenerPriority, which has a set of priorities. That is, at first, number of LOD, which has a low priority (SIMPLE_OBJECT), is sent. And then have those above, ending PANEL.
