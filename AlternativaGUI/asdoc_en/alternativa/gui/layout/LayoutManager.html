Classpublic class LayoutManager
InheritanceLayoutManager flash.events.EventDispatcher

Base class LayoutManager.

This manager responsible for positioning and sizing of containers.

Array of layers (guiLayers) contains workers layers and monitors add / delete objects in these layers.

Minimal scene size determined by the properties: minStageWidth, minStageHeight.
