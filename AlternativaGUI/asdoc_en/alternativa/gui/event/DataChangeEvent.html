Classpublic class DataChangeEvent
InheritanceDataChangeEvent flash.events.Event

DataChangeEvent class defines the event. This event are dispatched when data, which associated with the component, are change. This event are used by components: ListItemContainer, TreeItemContainer and DataProvider.
