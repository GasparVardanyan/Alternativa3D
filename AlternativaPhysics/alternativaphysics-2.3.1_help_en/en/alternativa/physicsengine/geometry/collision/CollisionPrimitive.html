public function raycast(origin:Vector3, vector:Vector3, epsilon:Number, normal:Vector3):Number

Calculates the intersection of the ray with the primitive.

Parameters
origin:Vector3 — Ray origin (at global coordinates).
 
vector:Vector3 — Direction vector of the ray on global coordinates. Vector can be any length (not zero).
 
epsilon:Number — Distance measurement error. The value does not exceed the absolute value of the specified error is zero.
 
normal:Vector3 — If intersection exist then normal of the primitive at the intersection point is saved in this parameter.

Returns
Number — If intersection exist then return the nonnegative time of the intersection point. Otherwise, it returns -1.
