raycast()method 
public function raycast(origin:Vector3, direction:Vector3, collisionMask:int, maxTime:Number, filter:IRaycastFilter, result:RayHit):Boolean

Checks whether the ray intersection with the physical geometry. The detector contains a set of primitives that are checked. In the case of multiple intersections, the method return intersection that nearest to the top of the ray.

Parameters
origin:Vector3 — Ray origin (at global coordinates).
 
direction:Vector3 — Direction vector of the ray in global coordinates. The length of the vector must be nonzero.
 
collisionMask:int — Intersections filter mask by groups.
 
maxTime:Number — Length of the inspected segment. Unit is one direction vector.
 
filter:IRaycastFilter — Filter of intersections.
 
result:RayHit — Parameter stores the information about the collision in the case of a positive test. Otherwise, the preservation of the initial data in the passed structure is not guaranteed.

Returns
Boolean — true if intersect exists. Otherwise, false.
